# 微分几何前置概念关系网：从拓扑空间到光滑流形

学习微分几何时，我最容易卡住的地方并不是某个公式本身，而是**这些概念到底处在哪一层**：哪些是拓扑层面的，哪些是光滑结构层面的，哪些只是坐标表达。混在一起以后，后面的切空间、张量、微分形式、联络、曲率都会变得像一团毛线。

所以这篇笔记的目标不是代替教材，而是搭一个可反复回看的框架：

1. 用一张关系网串起拓扑空间、连续、同胚、流形、图册和微分结构；
2. 把紧致、闭、完备、连通、弧连通等易混概念拆开；
3. 保留必要定义、公式、反例和口诀，让后续学习能接到同一棵树上。

一句主线先放在这里：

> **拓扑空间给出“连续”的语言；流形用“局部像 $\mathbb{R}^n$”把空间放进坐标；光滑图册要求坐标切换光滑，于是微积分可以在流形上无矛盾地进行。**

---

## 0. 三层视角：先分清对象所在的层

很多概念混乱，其实来自“层级错位”。我先把本文要用的三层视角列出来：

| 层级 | 关心什么 | 典型概念 | 关键词 |
| --- | --- | --- | --- |
| 拓扑层 | 什么叫邻近、连续、不撕裂 | 开集、邻域、闭包、连续、同胚、紧致、连通、Hausdorff | 拉回开集、拓扑不变量 |
| 光滑层 | 如何在空间上做微积分 | 坐标卡、光滑图册、坐标变换、微分结构 | 局部坐标、切换光滑 |
| 坐标表达层 | 如何把内在对象写成公式 | $f\circ\psi^{-1}$、坐标函数、坐标变换分量 | 表达依赖坐标，对象本身不依赖 |

这三层的关系是：

```text
拓扑层：先知道哪些映射连续，哪些空间拓扑上等价
   ↓
光滑层：在拓扑流形上挑选一套相容的光滑坐标
   ↓
坐标表达层：用坐标把内在对象写成普通多元函数
```

后面所有概念都可以放回这三层里定位。

## 1. 总关系图：从拓扑语言长出微分结构

```text
集合 X
│
└─ 指定开集族 𝒯
   │
   ▼
拓扑空间 (X, 𝒯)
   │
   ├─ 开集 / 闭集 / 邻域
   │    └─ 内部 A° / 闭包 Ā / 边界 ∂A
   │
   ├─ 连续映射：拉回开集仍开
   │    └─ 同胚：连续双射 + 逆连续
   │         └─ 保持拓扑性质：紧致、连通、Hausdorff 等
   │
   ├─ 关键拓扑性质
   │    ├─ 紧致：任意开覆盖可缩成有限子覆盖
   │    ├─ Hausdorff：两点能被不交开邻域分开
   │    ├─ 连通：不能拆成两个不交非空开集
   │    └─ 第二可数：有可数拓扑基
   │
   ▼
拓扑流形
   = Hausdorff + 第二可数 + 局部同胚于 R^n 开集
   │
   ▼
光滑流形
   = 拓扑流形 + 光滑图册
   │
   ├─ 坐标卡 (Oα, ψα)：局部地图
   ├─ 坐标变换 ψβ ∘ ψα⁻¹：重叠地图之间的换算
   └─ 微分结构：极大光滑图册 / 相容图册等价类
      │
      ├─ 光滑函数 f: M → R：看 f ∘ ψ⁻¹ 是否光滑
      └─ 乘积流形 M × N：乘积卡，维数相加
```

这棵树的记忆顺序可以压成一句话：

> **开集定邻近，连续看拉回；同胚保拓扑，图册给坐标；切换若光滑，微积分上岛。**

## 2. 拓扑空间：只谈“邻近”，先不谈距离

### 2.1 定义

拓扑空间是一个二元组 $(X,\mathcal{T})$：

- $X$：底层集合；
- $\mathcal{T}$：一族被指定为“开”的子集，满足：
  1. $\varnothing, X\in\mathcal{T}$；
  2. 任意多个开集的并仍开；
  3. 有限多个开集的交仍开。

这里的关键不是距离多远，而是**哪些集合被允许当作“局部观察窗口”**。在度量空间里，开球可以生成开集；但拓扑空间的定义更抽象，它不要求先有距离。

### 2.2 开集、闭集、邻域

| 概念 | 定义 | 记忆 |
| --- | --- | --- |
| 开集 | 属于 $\mathcal{T}$ 的集合 | 没有墙的广场，每点都有缓冲 |
| 闭集 | 补集是开集的集合 | 把极限边缘也收进来 |
| 邻域 | $N$ 是 $x$ 的邻域，当且仅当存在开集 $O$，使 $x\in O\subseteq N$ | 包着一块“露天小广场”的地盘 |

开集也可以用邻域刻画：

> $A$ 是开集，当且仅当 $A$ 是其中每一点的邻域。

这个刻画很有用，因为它把“整体开”转化成了“每个点周围都有一点余地”。

### 2.3 内部、闭包、边界

给定 $A\subseteq X$：

- **内部** $A^\circ$：$A$ 中包含的最大开集；
- **闭包** $\overline{A}$：包含 $A$ 的最小闭集；
- **边界**：

$$
\partial A=\overline{A}\setminus A^\circ
$$

记忆：

| 概念 | 比喻 |
| --- | --- |
| $A^\circ$ | 剥掉皮的净肉 |
| $\overline{A}$ | 连皮带肉 |
| $\partial A$ | 皮本身 |

一个常见直觉是：内部是“稳稳在里面”的点，闭包是“已经在里面或无限贴近它”的点，边界是“不管往哪边挪一点都能碰到里外”的点。

## 3. 连续与同胚：拓扑上“不撕裂”

### 3.1 连续：拉回开还是开

拓扑空间 $X,Y$ 之间的映射 $f:X\to Y$ 连续，定义为：

> 对任意 $Y$ 中开集 $V$，原像 $f^{-1}(V)$ 都是 $X$ 中开集。

口诀：**连续 = 拉回开还是开。**

点连续的说法是：对 $f(x)$ 的每个开邻域 $V$，存在 $x$ 的开邻域 $U$，使得：

$$
f(U)\subseteq V
$$

我用这句话记：**想困像，先困己。** 如果想把像控制在 $V$ 里，就能先在原空间找到一个足够小的 $U$。

这两种定义在普通拓扑空间里是等价的：全局的“拉回开集”可以推出每点附近的局部控制；反过来，处处有局部控制也能保证开集原像是开集。

### 3.2 同胚：拓扑意义下完全一样

同胚映射是：

1. 连续；
2. 双射；
3. 逆映射也连续。

直观上，同胚就是橡皮泥变形：**可以拉伸弯曲，但不能撕裂、粘合、打洞或填洞。**

同胚保持拓扑性质，例如紧致、连通、Hausdorff、第二可数等。它不保持长度、角度、面积、曲率，因为这些不是纯拓扑层面的信息。

一个很重要的警惕：

> 连续双射不一定是同胚，因为逆映射可能不连续。

所以“同胚”比“连续双射”多了一层保护：不仅点一一对应，而且开集结构也能来回无损搬运。

## 4. 四组拓扑性质：紧致、Hausdorff、连通、完备

### 4.1 紧致：无限问题能缩成有限检查

紧致定义为：

> 任意开覆盖都有有限子覆盖。

如果 $\{U_\alpha\}$ 是 $K$ 的开覆盖，即：

$$
K\subseteq\bigcup_\alpha U_\alpha
$$

那么紧致要求存在有限多个指标 $\alpha_1,\dots,\alpha_m$，使得：

$$
K\subseteq U_{\alpha_1}\cup\cdots\cup U_{\alpha_m}
$$

口诀：**任何覆盖都能瘦成有限层。**

在 $\mathbb{R}^n$ 中，Heine--Borel 定理说：

$$
K\subseteq\mathbb{R}^n\text{ 紧致}\Longleftrightarrow K\text{ 闭且有界}
$$

注意：这个“闭且有界”等价紧致是欧氏空间里的特殊好事，不能直接搬到任意拓扑空间。

常用事实：

- 连续像保持紧致；
- 紧致性是拓扑不变量；
- 在 Hausdorff 空间中，紧致子集一定是闭集。

### 4.2 Hausdorff：两点之间有防火墙

Hausdorff，也叫 $T_2$，定义为：

> 任意两个不同点 $x\neq y$，都存在不相交开邻域 $U,V$，使 $x\in U$、$y\in V$。

记忆：**两点之间能筑防火墙。**

它带来两个很重要的直觉：

- Hausdorff 空间中单点集是闭集；
- 序列、网或滤子的极限不会乱粘到多个点上；尤其在常见 Hausdorff 空间里，极限唯一。

严格地说，“单点集闭”只需要更弱的 $T_1$ 条件；Hausdorff 比 $T_1$ 更强，所以当然也有这个结论。

### 4.3 连通与弧连通

连通：空间不能拆成两个不相交非空开集的并。

弧连通：任意两点之间都有一条连续路径相连，也就是存在连续映射：

$$
\gamma:[0,1]\to X,
\quad \gamma(0)=x,
\quad \gamma(1)=y
$$

关系：

$$
\text{弧连通}\Longrightarrow\text{连通}
$$

但反过来不一定。典型反例是拓扑学家的正弦曲线：它连通，但不是弧连通。

记忆：

- 连通：不能分家；
- 弧连通：任意两点有路；
- 有路一定没分家，没分家不一定有路。

### 4.4 完备性：分析意义上的“没有洞”

完备性通常不是纯拓扑性质，而是度量空间里的性质。一个度量空间完备，意思是每个柯西列都收敛到空间内部的点。

对实数 $\mathbb{R}$，完备性有很多等价表述：

- 柯西列收敛；
- 确界原理；
- 区间套定理。

要分清：

- $\mathbb{R}$ 完备，但不紧致，因为无界；
- 闭区间 $[a,b]$ 在 $\mathbb{R}$ 中既完备又紧致；
- $\mathbb{Q}$ 不完备，也不紧致。

我把它们这样记：**完备管“洞”，紧致管“有限化”。**

## 5. 从拓扑流形到光滑流形

### 5.1 拓扑流形：局部像 $\mathbb{R}^n$

常见定义中，一个 $n$ 维拓扑流形通常要求：

1. Hausdorff；
2. 第二可数；
3. 每一点都有邻域同胚于 $\mathbb{R}^n$ 的某个开集。

也就是说，流形整体可以很弯、很绕、甚至不能用一张图摊平；但每个足够小的局部都像欧氏空间。

比喻：**地球不能用一张平面地图无失真铺开，但每个城市附近都可以画成一张平面地图。**

这里的 Hausdorff 与第二可数不是装饰。粗略地说，Hausdorff 防止点之间分不开，第二可数防止空间大到无法用可数张“基础地图”管理。很多教材会把它们写进流形定义里，正是为了排除病态空间。

### 5.2 坐标卡与图册

一个坐标卡是 $(O,\psi)$：

- $O\subseteq M$ 是开集；
- $\psi:O\to V\subseteq\mathbb{R}^n$ 是同胚；
- $V$ 是 $\mathbb{R}^n$ 中的开集。

一族坐标卡 $\{(O_\alpha,\psi_\alpha)\}$ 覆盖 $M$，就像一本地图册。

坐标卡的作用有两层：

1. 拓扑层：说明局部确实长得像欧氏开集；
2. 计算层：把流形上的点写成 $(x^1,\dots,x^n)$，让普通多元函数进入舞台。

### 5.3 坐标变换：地图重叠处的换算公式

如果两个坐标卡有重叠：

$$
O_\alpha\cap O_\beta\neq\varnothing
$$

那么从 $\alpha$ 坐标换到 $\beta$ 坐标的公式是：

$$
\Phi_{\beta\alpha}
=\psi_\beta\circ\psi_\alpha^{-1}
:
\psi_\alpha(O_\alpha\cap O_\beta)
\to
\psi_\beta(O_\alpha\cap O_\beta)
$$

这个映射发生在 $\mathbb{R}^n$ 的开集之间，所以可以谈普通多元微积分意义下的光滑性。

如果写成坐标：

$$
y=\Phi_{\beta\alpha}(x)
$$

其中 $x=(x^1,\dots,x^n)$，$y=(y^1,\dots,y^n)$。每个 $y^i$ 都可能依赖全部 $x^j$，所以坐标变换就是 $n$ 个 $n$ 元函数：

$$
y^i=y^i(x^1,\dots,x^n),\quad i=1,\dots,n
$$

要求“光滑”，就是要求这些分量函数都有任意阶连续偏导。

### 5.4 光滑图册与微分结构

光滑图册：覆盖 $M$ 的坐标卡族，并且任意两个重叠坐标卡之间的坐标变换都是 $C^\infty$。

相容：两个坐标卡或图册合在一起后，坐标变换仍然光滑。

微分结构可以理解为：

> 一个极大光滑图册，也就是把所有与当前图册相容的坐标卡都收进来。

这一步很关键。拓扑流形只说“局部像 $\mathbb{R}^n$”，光滑流形还要指定“哪些坐标卡之间的切换算光滑”。

记忆：

> **拓扑流形给地图，光滑图册给换图规则，微分结构收齐所有合法地图。**

### 5.5 为什么定义中只要求坐标卡是同胚？

刚定义流形时，我们还没有定义“流形之间的光滑映射”，所以不能一开始就说 $\psi_\alpha$ 是微分同胚，否则会循环定义。

因此定义先要求：

$$
\psi_\alpha:O_\alpha\to V_\alpha\subseteq\mathbb{R}^n
$$

是同胚，保证拓扑层面局部像欧氏开集。

等光滑结构建立之后，再把 $O_\alpha$ 和 $V_\alpha$ 都看作光滑流形，就能回头说：坐标映射实际上是局部微分同胚。

这不是定义偷懒，而是顺序问题：

```text
先有拓扑局部模型
→ 再用坐标切换定义光滑兼容
→ 最后才有流形间光滑映射与微分同胚
```

## 6. 两个基础例子：圆周与乘积流形

### 6.1 圆周 $S^1$ 是一维流形

单位圆：

$$
S^1=\{(x,y)\in\mathbb{R}^2:x^2+y^2=1\}
$$

它是 1 维流形，因为每个小弧段都像开区间。

为什么不能用一个坐标卡把整个 $S^1$ 同胚到 $\mathbb{R}$ 的开区间？一个直观理由是：$S^1$ 紧致，而 $\mathbb{R}$ 的非空开区间不紧致。同胚会保持紧致性，所以不可能。

一种常见做法是用四个半圆覆盖，例如按 $x>0,x<0,y>0,y<0$ 分成四张图，在每张图上投影到某个坐标轴得到 $(-1,1)$。

在重叠区，坐标变换会长得像：

$$
t\mapsto\sqrt{1-t^2}
$$

只要限制在对应开区间和正确分支上，它就是光滑的。

记忆：**圆不能一张地图摊平，但可以用几张局部地图拼起来。**

### 6.2 乘积流形：坐标拼起来，维数加起来

如果 $M$ 是 $m$ 维流形，$N$ 是 $n$ 维流形，那么 $M\times N$ 可以做成 $(m+n)$ 维流形。

坐标卡取：

$$
(U\times V,\phi\times\psi)
$$

其中：

$$
(\phi\times\psi)(p,q)=(\phi(p),\psi(q))
$$

维数公式：

$$
\dim(M\times N)=\dim M+\dim N
$$

例子：

- 圆柱面：$S^1\times\mathbb{R}$；
- 环面：$S^1\times S^1$。

## 7. 流形上的函数：对象本身 vs 坐标表达

流形上的光滑函数是：

$$
f:M\to\mathbb{R}
$$

它是一个内在对象，比如“温度场”。但要检查它是否光滑，需要放进坐标卡里看：

$$
f\circ\psi^{-1}:\psi(O)\subseteq\mathbb{R}^n\to\mathbb{R}
$$

如果对每张坐标卡，$f\circ\psi^{-1}$ 都是普通意义下的光滑函数，那么 $f$ 就是流形上的光滑函数。

这里要分清：

| 层次 | 对象 | 依赖坐标吗 |
| --- | --- | --- |
| 流形上 | $f:M\to\mathbb{R}$ | 不依赖，是内在对象 |
| 坐标里 | $F=f\circ\psi^{-1}$ | 依赖坐标，是表达式 |

换一个坐标卡，表达式会变，但函数本身没变。后面学向量场、微分形式、度量张量时，这个区分会反复出现：**对象在流形上，公式在坐标里。**

## 8. 易混概念对照表

### 8.1 紧致 vs 闭 vs 完备

| 概念 | 属于哪类结构 | 核心意思 | 典型提醒 |
| --- | --- | --- | --- |
| 闭 | 拓扑概念 | 补集开；常见空间里包含极限点 | $\mathbb{R}$ 在自身中闭，但不紧致 |
| 紧致 | 拓扑概念 | 任意开覆盖可取有限子覆盖 | 在 $\mathbb{R}^n$ 中等价于闭且有界 |
| 完备 | 度量概念 | 柯西列有归宿 | $\mathbb{R}$ 完备但不紧致；$\mathbb{Q}$ 不完备 |

一句话：**闭看边界收没收，完备看洞补没补，紧致看无限能不能有限化。**

### 8.2 连通 vs 弧连通

| 概念 | 问的问题 | 关系 |
| --- | --- | --- |
| 连通 | 能不能被开集拆成两块？ | 更弱 |
| 弧连通 | 任意两点之间有没有连续路径？ | 更强，推出连通 |

记忆：**有路一定没分家；没分家不一定有路。**

### 8.3 同胚 vs 微分同胚

| 概念 | 层次 | 要求 | 保持什么 |
| --- | --- | --- | --- |
| 同胚 | 拓扑 | 连续双射，逆连续 | 开集结构、紧致、连通等拓扑性质 |
| 微分同胚 | 光滑 | 光滑双射，逆光滑 | 光滑结构，允许微积分无歧义搬运 |

流形定义早期先用同胚，是因为光滑映射还没定义；有了微分结构后，坐标图才可被看成局部微分同胚。

### 8.4 拓扑流形 vs 光滑流形

| 概念 | 多了什么结构 | 能做什么 |
| --- | --- | --- |
| 拓扑流形 | 局部同胚于欧氏开集 | 能谈连续、同胚、紧致、连通 |
| 光滑流形 | 相容的光滑图册 / 微分结构 | 能谈光滑函数、切空间、微分形式等 |

拓扑流形像“有地图轮廓的地球仪”，光滑流形像“地图之间有光滑换算规则的地图册”。

### 8.5 流形上的对象 vs 坐标表达

| 例子 | 本体 | 坐标表达 |
| --- | --- | --- |
| 标量场 | $f:M\to\mathbb{R}$ | $f\circ\psi^{-1}$ |
| 点 | $p\in M$ | $\psi(p)=(x^1,\dots,x^n)$ |
| 坐标函数 | $x^i:O\to\mathbb{R}$ | 第 $i$ 个投影 |

口诀：**对象在流形上，公式在坐标里。**

### 8.6 图册 vs 微分结构

| 概念 | 作用 |
| --- | --- |
| 坐标卡 | 一张局部地图 |
| 图册 | 一组覆盖全空间且互相兼容的地图 |
| 极大图册 / 微分结构 | 所有与图册相容的合法地图全集 |

记忆：**图册是起步版本，微分结构是收齐所有合法地图后的版本。**

## 9. 常见坑：哪些话要加条件

下面这些句子很常用，但都带着隐含条件：

| 说法 | 应补的条件 / 解释 |
| --- | --- |
| 紧致等价于闭且有界 | 这是 $\mathbb{R}^n$ 中的 Heine--Borel 定理，不是任意拓扑空间的结论 |
| 紧致子集是闭集 | 需要 ambient space 是 Hausdorff |
| 单点集闭 | Hausdorff 可推出；更弱的 $T_1$ 已足够 |
| 连续双射就是同胚 | 不对，还需要逆映射连续；紧致到 Hausdorff 的连续双射才自动是同胚 |
| 完备是拓扑性质 | 一般不是，它依赖度量；同一个拓扑可能来自不同度量，完备性可能改变 |
| 一个流形就是局部像 $\mathbb{R}^n$ | 常见定义还要求 Hausdorff、第二可数；光滑流形还要光滑图册 |

这张表的作用是提醒我：**定理不是口诀，口诀只负责定位，条件才负责正确。**

## 10. 口诀与反例索引

| 概念 | 口诀 / 比喻 | 反例提醒 |
| --- | --- | --- |
| 开集 | 没有墙的广场；每点都有缓冲 | 拓扑不同，开集不同 |
| 闭集 | 补集开；边缘也收进来 | 闭不等于紧致：$\mathbb{R}$ 闭但不紧致 |
| 邻域 | 包含一块露天小广场的地盘 | 邻域本身不一定开 |
| 内部 | 剥掉皮的净肉 | 可能为空 |
| 闭包 | 连皮带肉 | 会补上贴边点 |
| 边界 | 皮本身 | $\partial A=\overline{A}\setminus A^\circ$ |
| 连续 | 拉回开还是开 | 不必先有距离 |
| 同胚 | 橡皮泥变形，不撕不粘 | 保拓扑，不保长度角度 |
| 紧致 | 覆盖能瘦成有限层 | $\mathbb{R}$、开区间不紧致 |
| Hausdorff | 两点之间有防火墙 | 非 Hausdorff 中极限可能不唯一 |
| 连通 | 不能分家 | 连通不一定弧连通 |
| 弧连通 | 任意两点有路 | 拓扑学家的正弦曲线提醒反向不成立 |
| 完备 | 柯西列有归宿 | 完备不等于紧致 |
| 流形 | 地图册拼成的空间 | 整体不一定一张图摊平 |
| 第二可数 | 基础积木只有可数块 | 常用来排除太大的病态空间 |
| 微分结构 | 合法坐标切换规则全集 | 同一拓扑流形可能有不同光滑结构 |

## 11. 最后一棵记忆树

```text
拓扑空间
├─ 开集语言
│  ├─ 闭集 = 开集的补
│  ├─ 邻域 = 包含开集缓冲
│  └─ 内部 / 闭包 / 边界
│
├─ 连续语言
│  ├─ f 连续 ⇔ f⁻¹(开) 开
│  └─ 同胚 = 连续双射 + 逆连续
│
├─ 拓扑性质
│  ├─ 紧致：开覆盖有限化
│  ├─ Hausdorff：点可分离
│  ├─ 连通：不能拆开
│  └─ 第二可数：可数基
│
└─ 流形语言
   ├─ 拓扑流形：局部同胚于 Rⁿ 开集
   ├─ 坐标卡：局部地图
   ├─ 坐标变换：ψβ ∘ ψα⁻¹
   ├─ 光滑图册：所有换图公式光滑
   ├─ 微分结构：极大光滑图册
   ├─ 光滑函数：f ∘ ψ⁻¹ 在坐标里光滑
   └─ 乘积流形：M×N，维数相加
```

如果以后继续往后学，我会把新概念接到这棵树下面：

```text
微分结构
└─ 切空间 / 余切空间
   └─ 张量 / 微分形式
      └─ 联络 / 曲率
```

本文先停在这里：把拓扑和光滑流形的地基打清楚。后面再遇到新的定义时，先问两个问题：

1. 这个概念属于拓扑层、光滑层，还是坐标表达层？
2. 它依赖哪些前置条件，哪些结论只在特殊空间里成立？

只要这两个问题不乱，微分几何后面的概念就不再是孤立名词，而是能接回同一张地图的节点。
